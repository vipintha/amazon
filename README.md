# Amazon E-Commerce SQL Analytics Project

## Project Overview
This project analyzes an Amazon-like e-commerce database using SQL to derive meaningful business insights related to sales, customers, inventory, and returns.  
The objective is to demonstrate real-world SQL skills required for a Data Analyst role.

---

## Business Problems Solved

### 1. Top Products Analysis
- Identified top 10 products based on total sales value
- Calculated total quantity sold and total revenue per product

### 2. Category Revenue Contribution
- Computed total revenue for each product category
- Calculated percentage contribution of each category to overall revenue

### 3. Customer Average Order Value (AOV)
- Calculated Average Order Value per customer
- Filtered customers with more than 5 orders

### 4. Monthly Sales Trend
- Analyzed monthly total sales for the last 1 year
- Compared current month sales with previous month sales

### 5. Inactive Customers
- Identified customers who registered but never placed an order
- Calculated time since customer registration

### 6. Least-Selling Category by State
- Determined the lowest-selling product category in each state
- Used window functions for ranking

### 7. Customer Lifetime Value (CLTV)
- Calculated total revenue generated by each customer
- Ranked customers based on CLTV

### 8. Inventory Monitoring
- Identified products with stock levels below a defined threshold
- Included warehouse and last restock date information

### 9. Payment Status Analysis
- Analyzed payment success, failure, and pending rates
- Calculated percentage distribution of payment statuses

### 10. Product Return Analysis
- Identified top 10 products with highest returns
- Calculated return rate as a percentage of total units sold

## SQL Concepts Used
- Joins (INNER, LEFT)
- Aggregations (SUM, COUNT, AVG)
- Common Table Expressions (CTEs)
- Window Functions (ROW_NUMBER, DENSE_RANK)
- Date functions
- Conditional logic (CASE WHEN)
